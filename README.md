# Duroob-Al-Shatat
​"دروب الشتات واليقين: ملحمة سودانية تُعرّي زيف الرخام أمام صلابة الطين. حين تهوي نجفة المنشية الكريستالية، تبدأ رحلة مرتضى الوجودية من ترف الخرطوم إلى دفء عريطل، ثم تيه الصحراء وزحام المنافي. رواية تختزل وجع النزوح وكبرياء العودة، لتثبت أن الوطن ليس جدرانًا تحمينا، بل هو يقينٌ يسكن القلوب، وقصة شعبٍ ينهض من الرماد ليبني مستقبله بالصبر والانتماء."
